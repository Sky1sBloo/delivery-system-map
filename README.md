# Delivery System Map
Map made in godot for (Deliverithm)[https://github.com/Sky1sBloo/delivery-system-frontend.git]

## Requirements
- Godot Engine
- Deliverithm

## Installation
1. Build as html5
2. Save to `src/static/map` in (Deliverithm)[https://github.com/Sky1sBloo/delivery-system-frontend.git]
